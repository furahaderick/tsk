# TSK: A CLI app for managing daily tasks

Manage your daily tasks inside the terminal

## Installation & Usage

### 1. Prerequisites
- Node.js
- MongoDB

### 2. Clone repository
```bash
git clone <url>
cd /path/to/repository
npm link
```
### 3. Create .env file
```env
MONGO_URI=<connection_string>
```

### 4. Run the app
```bash
tsk
```
